# Sales Dashboard Analysis – Excel Practical Exam

## Project Overview
This project is an interactive Excel Sales Dashboard created using a sales dataset containing 300 records. The dashboard provides insights into sales performance, profit analysis, regional sales distribution, product category performance, and customer trends.

The project demonstrates Excel skills including formulas, Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Dashboard Design.

---

## Dataset Information

**Dataset Name:** Sales_Data

**Total Records:** 300

**Fields Used:**

| Column Name | Description |
|------------|-------------|
| Order ID | Unique order identifier |
| Order Date | Date of order |
| Region | Sales region |
| Category | Product category |
| Product Name | Product sold |
| Customer Name | Customer name |
| Quantity | Quantity sold |
| Sales | Sales amount |
| Discount | Discount applied |
| Profit | Profit earned |
| Profit Margin | Profit percentage |
